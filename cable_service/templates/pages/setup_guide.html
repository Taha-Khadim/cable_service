Cable Service Setup Guide

Initial Setup Required

Follow these steps to set up your Cable Service Management system:

1Install the App

Run this command in your terminal:

bench --site [your-site-name] install-app cable_service
2Migrate Database

Create the database tables:

bench --site [your-site-name] migrate
3Create Sample Data

After migration, create some sample packages:

  • News Package - $15/month
  • Sports Package - $25/month
  • Kids Package - $10/month
  • All-in-One Package - $45/month
4Access the System

Once setup is complete:

  • Visit /login for seller/admin access
  • Use Administrator/admin for initial login
  • Access the dashboard at /dashboard
After Setup

Your system will include:

  • Package management
  • Customer registration
  • Payment tracking
  • Service status management
